This sub-folder holds the PDF versions of the Getting Started Guides produced during the Kickstarter in 2013

Since theses were only available on the partyrobotics website, we were able to recover them using the Wayback Machine and am uploading them here for posterity and to able to reference them in future.

Great thanks to the Internet Archive <3
