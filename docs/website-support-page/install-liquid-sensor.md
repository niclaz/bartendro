##  Installing a Liquid Level Sensor 

If you've purchased a liquid level sensor kit this is the place that will show you how to install it onto your Bartendro Dispenser.

Tools: #2 Philips Screwdriver
Relevant Parts: Liquid Level Sensor Kit (part number: MPX5010DP)

### Step 1 — Prepare the Dispenser and Parts
- Remove the clear dome with the hub from the dispenser so that you can rest the Dispenser flat for easier installation.
- Make sure you have all the parts that you need including a Philips screwdriver.
- Use something rigid like a credit card to apply pressure on top of the pins so that they go in straight without bending. Be gentle as the pins can be delicate.


### Step 2 — Install the Sensor
- Line up the pins of the sensor with the header making sure they are correctly aligned.
- Use something rigid like a credit card to apply pressure on top of the pins so that they go in straight without bending. Be gentle as the pins can be delicate.


### Step 3 — Screw the Sensor Into Place
- Attach the included screws with a #2 Philips screwdriver. Be careful to not tighten too much. If you over-tighten the case may crack and destroy the sensor. Err on the lighter side.
- To make use of this feature, go to the Admin > options menu and check 'use liquid sensors'. This will require that you press the 'save and reboot' button at the bottom of the page.

### Step 4 — Install the Black Tubing
- The provided black tubing and barbed fitting serve to provide a better seal for the clear tubing. They also protect the sensor's barb from breaking if the clear tubing is pulled on too hard.
- To make use of this feature, go to the Admin > options menu and check 'use liquid sensors'. This will require that you press the 'save and reboot' button at the bottom of the page.
- After a few seconds, your bot will reboot and a new tab will appear in the Admin menu called 'liquid level'. This tab will allow you to calibrate the sensor as necessary. Follow the on-screen instructions.
