### How to Get to Bartendro's Admin Menu

To change your ingredient lineup, create new drinks, default sizes among many other things, you'll need to access the admin section of Bartendro.

### Step 1 — The Main Menu

- This is the main Menu where you can select from your choice of drinks. You are only shown what is available to be made.
- Clicking, or tapping on the logo in the top right corner will get you behind the scenes into the admin menu where the setup of many things is done.
- The default user is bartendro and password is boozemeup.
- You can change this password in the Options menu.

### Step 2 — Shots Menu

- The second tab at the top allows you to get into the Shots Menu where you can dispense shots or custom drinks on the fly be dispensing one shot at a time.
- A test dispense is perfect for priming empty tubing. It usually takes 2 presses of the 10mL dispense to fill up fresh tubing with minimal booze waste. The value is adjustable in the Options tab.

### Step 3 — The Admin Home Page - Dispensers
- This page allows you to see all your connected dispensers and control them. Newer dispensers have the ability for the motors to move backwards as well!
- A test dispense is perfect for priming empty tubing. It usually takes 2 presses of the 10mL dispense to fill up fresh tubing with minimal booze waste. The value is adjustable in the Options tab.
- You can select each ingredient and change it to be something else. If you want to use a booze that's not listed, you'll have to go to the booze tab and add the new ingredient first.
- Once you've made your selections, don't forget to press save at the bottom.

### Step 4 — Admin Menu - Drinks
- Modify an existing drink, or create a new one.

### Step 5 — Admin Menu - Booze
- Modify an existing ingredient, or create a new one.

### Step 6 — Admin Menu - Options
- This is where all the advanced options are.
- Edit the default sizes of pours and turn features on and off.
