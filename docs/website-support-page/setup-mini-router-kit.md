## Setting up your Bartendro miniRouter Kit 

This guide will show you what you need to do to get up and running and dispensing cocktails as quickly as possible. It shouldn't take more than 15 minutes.

### Step 1 — Setting up your Bartendro miniRouter Kit
- To get started, unpack your components and make sure you have everything you need to get started.
- As a bare minimum, you need a Raspberry Pi (RPI), a Router / miniRouter board, an SD card with properly installed firmware, a USB WiFi dongle, and a compatible 24V power supply.

### Step 2 — Attach Raspberry Pi
- Attach Standoffs to the Raspberry Pi's two mounting holes.

### Step 3 — Attach miniRouter to Raspberry Pi
- Make sure that the header from the miniRouter is perfectly aligned with the RPI otherwise you could cause damage.
- Once the miniRouter board is resting on the RPI, screw it in with the provided screws.

### Step 4
- Install the USB WiFi Dongle into either of the USB ports.
- Install the SD card with the Bartendro firmware. To install firmware on a fresh SD card, follow this guide.

### Step 5
- Plug in your dispensers. Order shouldn't matter, but it's easier to keep track if you start with Port 1 and work your way up.
- With the switch in the off position, plug the power supply in and turn the switch on.

### Step 6 — Boot up and dispense your first cocktail!
- You'll need to wait about a minute for the operating system to boot. The dispensers may be blue or green, or off because they're not getting proper signals, but once the bot is booted and ready, you'll see a color changing patten on each dispenser.
- You're almost ready to reward yourself with a tasty cocktail. View our next guide on how to get around the interface.
