###  Password Recovery 

If you've changed the password to access Bartendro's interface, but have forgotten it. This guide will show you how to retrieve it.

### Step 1 — Go to a web browser
- At the end of the URL to the admin screen, type: lost-passwd
- You'll notice that if you go to that address while connected through WiFi, you will get this screen. WiFi access to this page is disabled for security reasons. To retrieve the password, the Raspberry Pi must be wired to the same network that your interface device is on.

### Step 2
- The Raspberry Pi needs to be connected to a wired Ethernet hub or router so that you can connect with your local network.
- Make sure to use the Ethernet port on the side of the Raspberry Pi and not one of the dispenser ports!
